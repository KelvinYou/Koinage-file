Create wallet

Request method : get

Api key : http://1.32.63.61:8051/api/v1/wallet/CreateApiKey

Paramater :1) Type=Omni, LTC, BCH, ETH 2)Label= user email

example : http://1.32.63.61:8051/api/v1/wallet/CreateApiKey?type=LTC&label=test@test.com

Register account

Request method : post

Api key : http://1.32.63.61:8491/api/v1/register

Paramater

nickname 昵称，2-16位 * 中文 \u4e00-\u9fA5 * 日文 \u0800-\u4e00 * 韩文 \uac00-\ud7ff

pwd 8~16 0-9a-zA-Z no special char

safe_pwd 8~16 0-9a-zA-Z no special char

email

emailCode 4 digit code which will receive by email

example : http://1.32.63.61:8491/api/v1/register?nickname=bro&pwd=June0612&safe_pwd=Card0912&email=test@test.com&emailCode=123456

Sample 1: { "code": 0, "msg": "success", "result": null } Sample 2: { "code": 40008, "msg": "account existed", "result": null } Sample 3: { "code": 40003, "msg": "invalid password format", "result": null }Sample 4: { "code": 40004, "msg": "invalid password format", "result": null }

Withdraw

Request method : get

Api key : http://1.32.63.61:8051/api/v1/wallet/Withdraw

Paramater

Params 1 = api_key (api_key is different for each currency and each client)

Params 2 = amounts (Numberic)

Params 3 = from_addresses (Varchar) the address u get from the create wallet api key

Params 4 = to_addresses (Varchar)

Params 5 = network (Exp: BTC/USDT/ETH/JE)

example : http://1.32.63.61:8051/api/v1/wallet/Withdraw?api_key=xxxxx&amounts=0.0001&from_addresses=161UAMjnnKkX43ouDCH1iqJH1wWgQWC5gk&to_addresses=161UAMjnnKkX43ouDCH1iqJH1wWgQWC5gk&network=BTC
